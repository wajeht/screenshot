Capture screenshots of any URL.

Basic Usage

<img src="https://screenshot.jaw.dev?url=https://github.com" />

Preview on Hover

<script>
  document.addEventListener("DOMContentLoaded", () => {
    const preview = document.querySelector(".preview");
    const img = preview.querySelector(".preview-window");

    preview.addEventListener("mouseenter", () => {
      img.style.display = "block";
    });

    preview.addEventListener("mouseleave", () => {
      img.style.display = "none";
    });
  });
</script>

<div class="preview">
  <a href="https://github.com/">Github</a>
  <img
    class="preview-window"
    src="https://screenshot.jaw.dev?url=https://github.com/"
    loading="lazy"
    alt="Preview"
  />
</div>

<style>
  .preview {
    position: relative;
    display: inline-block;
  }

  .preview-window {
    position: absolute;
    top: 100%;
    left: 0;
    display: none;

    max-height: 200px;
    max-width: 300px;

    width: 300px;
    height: auto;

    overflow: hidden;
    border: 1px solid #ccc;
    background: #fff;
  }
</style>

Options

url
URL to screenshot (required)
preset
thumb, og, twitter, square, mobile, desktop
width
custom width (max 1920)
height
custom height (max 1920)
full
set to true for full page

Examples

/?url=github.com
/?url=github.com&preset=twitter
/?url=github.com&width=800&height=600
/?url=github.com&full=true
